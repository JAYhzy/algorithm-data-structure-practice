# 项目说明

## 仓库定位

本仓库用于集中整理本科阶段算法与数据结构练习，方便从语言分类仓库中单独展示算法能力。

## 内容分类

- `cpp-algorithm-practice`：C/C++ 算法练习，包括排序、查找、数据结构和高精度阶乘等。
- `cpp-CL_Algorithm`：C++ 算法实现，包括多种排序、二分查找等。
- `cpp-CL_Datastruct`：C++ 数据结构实现，包括顺序表、链表、栈、树、KMP、AVL、哈夫曼树等。
- `java-stack-queue`：Java 栈、队列、链表练习。
- `java-binary-tree`：Java 二叉树和堆相关练习。
- `java-sorting`：Java 排序算法练习。
- `java-search-tree-map-set`：Java 搜索树、Map、Set 相关练习。

## 学习重点

- 排序算法
- 查找算法
- 链表、栈、队列
- 树、堆、搜索树
- 字符串匹配
- 多语言算法实现能力

## 运行说明

C/C++ 部分可使用 Visual Studio、CLion、CMake 或 GCC/MinGW 编译。Java 部分可使用 IntelliJ IDEA / Eclipse 打开源码目录运行。

## 整理说明

仓库只保留源码和必要项目配置，过滤了构建产物和 IDE 缓存。部分内容也存在于 C++ 或 Java 分类仓库中，本仓库作为算法专题索引与展示。
